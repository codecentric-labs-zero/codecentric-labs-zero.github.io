Overview

We document our experience with different parts of our tool chain and framework stack here.

Things we use

We found the following tools and frameworks useful.

axios

Simple promise based HTTP client. Out of the box request http methods(get,post,put,delete,patch...).

material ui

Highly customizable material design components for React Native! Action Button, Avatar, Button, Card, Checkbox, Dialog, Divider, Drawer, Icon, Icon toggles, List item, Radio button, Subheader, Toolbar.
